# ChiroHealth — Premium Chiropractic Website Template

Warm, premium single-page template for chiropractic practices. Full-width cinematic hero, earthy terracotta/cream palette, dark/light mode, parallax CTA band, elegant serif typography.

## Colour Palette
- Primary: Terracotta (#B85C38 light / #D4845A dark)
- Accent: Gold (#C9A96E)
- Background: Warm cream (#FBF8F4)

## Fonts (CDN)
- Cormorant Garamond (headings)
- Nunito Sans (body)
- Bootstrap Icons v1.11.3

## Customise
Replace: practice name, phone, email, address, team, testimonials, pricing, hours.

Free for personal and commercial use.

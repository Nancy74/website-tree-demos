=============================================================
 TIPS FOR BEST RESULTS
=============================================================

• Minimum size: 800px wide for most images, 1400px+ for heroes
• Format: JPG is fine (smaller file size), WebP even better
• Keep filenames EXACTLY as listed above (lowercase, hyphens)
• Shoot or source on a warm cream/white background where possible
  to match the site palette
• The site uses object-fit:cover so images crop automatically —
  put the most important content centred

=============================================================
 HOW TO SWAP (3 steps)
=============================================================

1. Name your image exactly as listed (e.g. priya-portrait.jpg)
2. Drop it into the /img/ folder (replacing any placeholder)
3. Reload the page — done. No code changes needed.

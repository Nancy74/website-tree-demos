# Images

Place your website images here. Recommended files:

- `hero.jpg` — Hero section main image (min 1040x1120px)
- `about.jpg` — About section image (min 800x1000px)
- `team-1.jpg` to `team-4.jpg` — Team member photos (min 600x640px, square or portrait)
- `og-image.jpg` — Social media sharing image (1200x630px)
- `favicon.png` — Browser tab icon (32x32px or 64x64px)

Tip: Optimise images before uploading using tools like:
- TinyPNG (tinypng.com)
- Squoosh (squoosh.app)
- ImageOptim (Mac)

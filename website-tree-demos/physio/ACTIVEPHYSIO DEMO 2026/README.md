# ActivePhysio — Physiotherapy & Sports Massage Website Template

A modern, responsive, single-page website template designed for physiotherapy practices and sports massage clinics. Features dark/light mode toggle, smooth animations, and mobile-first responsive design.

## Features

- **Dark / Light Mode** — Sun (☀) and Moon (🌙) toggle with localStorage persistence
- **Fully Responsive** — Desktop, tablet and mobile optimised
- **Smooth Animations** — Scroll-reveal, hover transitions, hero entrance animations
- **Accessible** — Semantic HTML, ARIA labels, keyboard-navigable FAQ
- **SEO Ready** — Meta tags, semantic structure, descriptive alt text
- **No Build Tools** — Pure HTML, CSS and vanilla JavaScript

## Project Structure

## Quick Start

1. Download/clone the project folder
2. Open `index.html` in your browser — works immediately
3. Replace placeholder content (see below)
4. Deploy to Netlify, GitHub Pages, or any static host

## Customisation

| Placeholder | Replace With |
|---|---|
| `ActivePhysio` | Your practice name |
| `020 7946 1234` | Your phone number |
| `hello@activephysio.co.uk` | Your email |
| `45 Wellness Lane, London SW4 7AB` | Your address |
| Team names & photos | Your actual team |
| Testimonial quotes | Real patient testimonials |
| Pricing (£60, £48, £55) | Your actual fees |
| Opening hours | Your hours |

## Colour Palette

- **Primary:** Teal (#0E918C light / #2BC4BE dark)
- **Accent:** Warm coral (#E8643A)
- Edit CSS variables in `[data-theme="light"]` and `[data-theme="dark"]` sections
